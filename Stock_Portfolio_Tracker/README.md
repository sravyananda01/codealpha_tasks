Stock Portfolio Tracker

Description

A Python application that calculates the total value of a stock portfolio based on user input and predefined stock prices.

Features

- User enters stock names and quantities
- Calculates investment value
- Displays total portfolio value
- Saves summary to a text file

Technologies Used

- Python

Concepts Used

- Dictionaries
- Loops
- Input/Output
- File Handling
- Arithmetic Operations
