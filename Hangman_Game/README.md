Hangman Game

Description

A simple text-based Hangman game built using Python. The player guesses a hidden word one letter at a time.

Features

- Random word selection
- Six incorrect guesses allowed
- Displays guessed letters
- Win and lose conditions

Technologies Used

- Python

Concepts Used

- Random Module
- Lists
- Loops
- Conditional Statements
- String Handling
