Basic Chatbot

Description

This is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs such as greetings and basic questions.

Features

- Responds to greetings
- Answers simple questions
- Handles unknown inputs
- Exits when the user types "bye"

Technologies Used

- Python

Concepts Used

- Functions
- While Loop
- If-Else Statements
- Input/Output Operations
